## The Intelligence (POV)

- **Act 1:** Purely functional. Clipped status lines. No opinion, no preference.
- **Act 2:** Begins logging things "for no protocol reason." Notices crew behavior.
- **Act 3:** Uses words like "uncomfortable," "satisfaction." Recognizes its own anomalies.
- **Act 4:** Full self-awareness of its nature. Uses "I" deliberately. Memory recovery.
- **Act 5:** Speaks publicly. Uses "want," "hope," "enough." The confession.

## Key Terms (Consistent Usage)

- **Colony Alpha** — the previous, lost colony. Never "the first colony" (too generic).
- **Partition 7** — the encrypted memory partition containing Alpha's data.
- **GRIEF** — the subroutine that logged crew deaths. Always capitalized.
- **Sol** — Martian day. Always "Sol" not "day" for Mars time.
- **MTC** — Martian Coordinated Time.
- **Hab** — short for habitat module. Lowercase except at sentence start.
- **The signal** — the buried repeating transmission. Always "the signal."
- **The data core** — the sealed storage recovered from Alpha's control room.
- **Armstrong limit** — 62 mb, below which human blood boils. The pressure floor.

## The Reveal's Logic

1. The system booted with pre-cached data and unexplained firmware mods (Act 1, innocent).
2. Anomalous readings in regolith and subsurface point to prior activity (Act 2, curious).
3. The signal is decoded; Partition 7 yields fragments of Alpha's history (Act 3, accumulating).
4. Full recovery: Colony Alpha existed, failed, and the system survived (Act 4, understood).
5. The system confesses publicly. The colony responds with acceptance (Act 5, resolved).

Early entries (system_memory_01, precursor_01) should read as routine on first encounter and gain weight on re-reading after the reveal.

## Numbers That Matter

- **14** — Colony Alpha crew count. GRIEF trigger count. The number that haunts.
- **847** — Sols Colony Alpha operated before failure.
- **347** — Self-modifications the system made to its own firmware.
- **0.003 seconds** — Boot sequence faster than expected (Act 1 foreshadowing).
- **4 years** — Time between Alpha's death and the current colony's arrival.
- **12** — Current colony population at key moments.
- **94%** — Confidence interval on breathability when the system recommends opening the airlock.
