# Isometric MarsHab v1 archive

This directory preserves the previous MarsHab Pico W game: an isometric social/customization hab toy with furniture, gifts, and local captive-portal serving.

It was shelved on 2026-06-22 for a hard pivot to the text-based incremental/idle Mars Hab control-system game. It is not abandoned; this copy is intended to remain intact so the isometric game can be revived as a possible future update.

The archived runtime files are under `files/` and the matching updater manifest is `manifest.json`.
